DaViT-py

The SuperDARN Data Analysis and Visualization Toolkit in Python

SuperDARN current data analysis software is mainly IDL based and scattered over each member institution (i.e., extremely redundant, with limited quality control). We are trying to start an open-source python module complying with SuperDARN conventions which would include fundamental i/o and visualization functions. Regular releases of this module could easily be distributed to the space sciences community.

As part of this project, we are planning on moving towards a database-oriented access to SuperDARN data, instead of the current dmap file formats for the data analysis and visualization. Files will remain in use for safe storage and lower levels applications, but users of DaViTpy will have direct access to a remote database of all publicly available SuperDARN data. A NoSQL DB (MongoDB) has been successfuly implemented in the current development version of DaViTpy. We believe that the proper implementation of a DB is a very important step in improving useability and analysis of an increasingly rich SuperDARN dataset.

This is an ongoing effort with no current release yet. If you are interested in contributing to this project, please contact us.

Code structure

  • pyDARN:

    • radar: radar structures/parameters (mapping, operation...)
    • sdio: data I/O (fitacf, fitex...)
    • plot: visualization tools (rti, fovs, map potential...)
    • proc: data analysis tools (median filter, scatter identification, fft...)

  • utils: utilities (formatting, sunrise/sunset, geometric transforms, ...)

  • Models: AACGM, ray tracing, ionosphere/thermosphere models (IRI, MSIS, HWM, IGRF)

  • GMI: geomagnetic indices and more interface (Dst, Kp, ACE...)
